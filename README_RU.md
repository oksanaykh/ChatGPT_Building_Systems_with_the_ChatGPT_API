# Построение систем с помощью ChatGPT API --- отчет по курсу

Этот репозиторий содержит структурированный отчет по курсу **"Building
Systems with the ChatGPT API"** от **DeepLearning.AI и OpenAI**.

Отчет описывает архитектурные подходы и лучшие практики разработки
систем на базе **LLM** с акцентом на **безопасность, точность и
масштабируемость**.

------------------------------------------------------------------------

# 1. Введение в LLM

LLM --- модели, обученные предсказывать **следующий токен**.

## Base LLM vs Instruction‑tuned

  -----------------------------------------------------------------------
  Характеристика          Base LLM                Instruction‑tuned
  ----------------------- ----------------------- -----------------------
  Обучение                Предсказание следующего Fine‑tuning на
                          слова                   инструкциях

  Оптимизация             Supervised Learning     RLHF

  Поведение               Продолжает текст        Следует инструкциям
  -----------------------------------------------------------------------

------------------------------------------------------------------------

## Токенизация

LLM работают с **токенами**.

Пример:

`lollipop → l + oll + ipop`

Эмпирическое правило:

-   **1 токен ≈ 4 символа**
-   **1 токен ≈ ¾ слова**

1.  не хранить ключи в коде
